# AI Contract Template Generator

Upload a contract document — AI extracts every dynamic field, assigns field types, and populates a template editor in seconds instead of hours.

## What it does
1. **Upload** a PDF or DOCX contract
2. **AI parses** the document, identifying all dynamic fields (dates, names, amounts, places, signatures)
3. **Auto-classifies** field types (DatePicker, Textbox, Textarea, Dropdown, etc.)
4. **Review screen** lets you verify, rename, retype, add, or remove fields before applying
5. **One-click template** — fields become interactive chips inside a rich-text editor

## Tech
- Built with Claude (AI coding assistant)
- Bootstrap 5 + vanilla JavaScript
- Responsive rich-text editor with toolbar
- Multi-step modal flow (upload → processing → review → template)
