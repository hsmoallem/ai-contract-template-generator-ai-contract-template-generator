# AI Contract Template Generator

**Upload a contract → AI extracts fields → Ready-to-use template in seconds.**

## What it does

1. **Upload** a PDF or DOCX contract document
2. **AI parses** the document, identifying every dynamic field — dates, names, amounts, locations, signatures
3. **Auto-classifies** each field as the correct type: DatePicker, Textbox, Textarea, Dropdown, Radio, Checkbox, FileUploader, etc.
4. **Review screen** lets you verify, rename, retype, add, or delete fields before applying — AI does the heavy lifting, you stay in control
5. **One-click template** — approved fields become interactive chips inside a rich-text editor, composing a full contract template automatically
6. **Multi-country support** — template metadata (country, office) drives context-aware field suggestions (UAE, KSA, Qatar, Kuwait, Bahrain, Oman)

## Key Features

| Feature | Description |
|---------|-------------|
| ✦ AI Field Extraction | Parses PDF/DOCX contracts and identifies 20-40+ dynamic fields automatically |
| ✦ Intelligent Field Typing | Fields classified by type based on content and context, not just regex |
| ✦ Review & Refine | All AI-detected fields shown with inline editing — rename, retype, remove, or add |
| ✦ Rich-Text Editor | Full toolbar (bold, italic, lists, tables, indentation, alignment) with field chips |
| ✦ Field Chips | Dynamic fields appear as interactive colored chips inside the editor — visually intuitive |
| ✦ Processing Animation | Step-by-step progress: upload → parse → identify → classify → generate |
| ✦ Validation | Required field and label checks before save |
